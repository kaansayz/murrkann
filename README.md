# Cafe POS System

## Sistem Amacı
Restoran yönetimini kolaylaştırmak için geliştirilen bir POS sistemidir. Kullanıcıların hızlı ve etkili bir şekilde sipariş almasını, masaları yönetmesini ve muhasebe işlemlerini yapmasını sağlar.

## Kısa Teknik Açıklama
Bu uygulama, Python ve Flask kullanılarak geliştirilmiştir. Web tabanlıdır, böylece kullanıcılar tarayıcı üzerinden erişim sağlayabilirler. Uygulama, dinamik bir sayfa yapısına sahip olup, veri dosyaları JSON formatında depolanır.

## İleriye Dönük İş Planı
- Masa yönetim sisteminin geliştirilmesi
- Ürün ve adisyon yönetimi modüllerinin eklenmesi
- Kullanıcı arayüzünün iyileştirilmesi ve daha fazla özellik eklenmesi.
